I write content for this site in Obsidian, and convert it to a static site with Quartz. I do this for a few reasons:

1. I've grown rather fond of Obsidian. For whatever reason, I've found it greatly increases my productivity when keeping notes, and notes have turned out to be pretty core to my process as a developer. 
2. There's no need for any dynamic content on my personal site, so a static site generator makes a great deal of sense.
3. Writing my own static site generator would not be a good use of my time when somebody else has already invested a lot of time in writing one.

There is a small downside; I don't currently have a spellchecker enabled in Obsidian. 

I'm serving it via CloudFlare Pages
